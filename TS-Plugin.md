# TunerStudio Plugin

The rusEFI TunerStudio plugin is part of [rusEFI Online](Online) suite.

The rusEFI TunerStudio plugin facilitates tune and log uploads.

The rusEFI Plugin allows you to continuously upload the tune as you edit it. If you are offline, tunes are saved into the Outbox folder and get uploaded once you gain connectivity.

[rusEFI Online Plugin Alpha version demo video](https://www.youtube.com/watch?v=PozvQICdHWQ)

rusefi_ts_plugin_launcher.jar is included in all bundles.
