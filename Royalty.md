# Royalty

There are no royalty obligations if you follow [the GPLv3 license](https://github.com/rusefi/rusefi/blob/master/license.txt).

You can fabricate hardware and sell it for profit without paying royalties as long as you follow [the GPLv3 license](https://github.com/rusefi/rusefi/blob/master/license.txt).

## Donations

rusEFI LLC definitely has expenses so your donations would be appreciated! One-time, monthly, or a fixed amount per board would all help. See [How to Help rusEFI](HOWTO-help-rusEFI#last-but-not-least).
